# VideoCutEditor icon

`VideoCutEditor-icon-source.png` is the ChatGPT Image output supplied by the repository owner. Its checkerboard was rendered into the RGB pixels rather than stored as transparency.

Run `scripts/New-AppIconAssets.ps1` to crop the icon tile, apply a rounded alpha mask, restore transparent padding, and generate the ICO and WinUI PNG assets. The transformation preserves the supplied icon design and does not redraw it.
