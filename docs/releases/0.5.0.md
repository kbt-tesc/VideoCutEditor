# VideoCutEditor 0.5.0

MP4/WebMの書き出し選択と、必要な場合だけ音声をAACまたはOpusへ変換する機能を追加したリリースです。

## 主な変更

- Re-encodeでMP4とWebMの書き出しに対応しました。
- Fast copyでは映像をコピーできる組み合わせに限り、MP4/WebMのコンテナを切り替えられます。
- 出力形式に合わない音声だけをAACまたはOpusへ自動変換できます。
- 音声の再エンコードを明示的に選択でき、音声ビットレートを設定できます。
- WebMのOpus音声ではVBR/CBRを選択できます。
- 音声設定欄の表示条件、幅、行配置、縦位置を調整しました。
- 複数クリップは登録したタイトルと選択形式に応じて`.mp4`または`.webm`で一括書き出しされます。

## 配布物

- `VideoCutEditor-0.5.0-win-x64-setup.exe`: 現在のWindowsユーザー向けインストーラー。管理者権限は不要です。
- `VideoCutEditor-0.5.0-win-x64.zip`: インストール不要のPortable版です。
- 各配布物の`.sha256`ファイル: ダウンロード後の整合性確認に使用できます。

## 注意事項

- 現在の配布物はコード署名されていないため、Windowsから発行元不明の警告が表示される場合があります。
- ffmpegとffprobeは同梱していません。PATHから自動検出するか、設定画面で指定してください。
- Fast copyでコンテナを切り替えられるかどうかは、入力映像・音声のコーデックとストリーム構成によって変わります。
