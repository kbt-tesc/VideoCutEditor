# VideoCutEditor 0.5.1

書き出し中の進捗確認とINFOウィンドウの使いやすさを改善した更新です。

## 主な変更

- 書き出し開始時にINFOウィンドウを自動表示します。
- 映像処理では処理済みフレーム数と推定総フレーム数を表示します。
- 音量解析中は音声用プログレスバーへ切り替わります。
- INFOウィンドウを「エンコーダー・メディア情報」「書き出しログ」「書き出し進捗」の固定順に整理しました。
- ウィンドウの高さを変えたときは書き出しログだけが伸縮します。
- 書き出しログへ既定ONの自動スクロールを追加しました。設定は保存されません。

## 配布物

- `VideoCutEditor-0.5.1-win-x64-setup.exe`: 現在のWindowsユーザー向けインストーラー。管理者権限は不要です。
- `VideoCutEditor-0.5.1-win-x64.zip`: インストール不要のPortable版です。
- 各配布物の`.sha256`ファイル: ダウンロード後の整合性確認に使用できます。

SHA-256:

- Portable ZIP: `384abe74807bb67bf4492fa9f7fc0406b427d327f0549d7f63288c169bbf688f`
- Installer: `62c6ee2a2bbedb580812fd23ba3a0ac42d9d6f0c7fc3f919fc68d5a417e0cfe5`

## 注意事項

- 現在の配布物はコード署名されていないため、Windowsから発行元不明の警告が表示される場合があります。
- ffmpegとffprobeは同梱していません。PATHから自動検出するか、設定画面で指定してください。
